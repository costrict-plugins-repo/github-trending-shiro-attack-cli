本版本为 **5.1.1**，主要交付 GUI 界面重设计、jEG/JMG 集成增强，以及多个回显和内存马修复。

### 新增

- **GUI 全新界面**：现代 CSS 主题（`app.css`）、Header/POST Body 并排布局、对齐网格重排
- **JMG 内存马生成 Tab**：新增 pass/path/key 输入字段，支持参数化生成
- **CLI `--jeg` 标志**：`exec` 命令支持第三方 jEG 回显生成器
- **CLI `--dynamic-memshell` 标志**：运行时 Javassist 编译内存马源码
- **jEG 回显增强**：MODE_CMD + Authorization 明文头，明文/Base64 双格式响应解析
- **Gadgets.createTemplatesImpl(byte[])**：支持直接传入字节码构造 TemplatesImpl

### 修复

- GUI 指定 Key 爆破在多 Shiro 场景下 `multiShiroMode` 硬编码 `false` 导致误判（Closes #89）
- GodzillaFilter/Servlet/TomServlet `parseObj` coyote.Request fallback + null guards
- `classifyHttpResponse` 回显标记检测优先级修正
- `coerceRememberMeCookieHeader` 接受 Base64 格式 payload
- 消除启动时 SLF4J / MLog 日志噪音
- 移除 echo 调试日志和 rememberMe 原始打印

### 升级说明

- 推荐下载 Release 中的 **`bundle.zip`**，解压后直接运行
- GUI 界面有较大变化，功能操作方式不变
